## **สรุปภาพรวมทั้งหมด:**
```
Application Layer:
┌──────────────────────────────────────┐
│ Controller                           │
│ - รับ/ส่ง HTTP (ใช้ DTO)             │
└──────────────────────────────────────┘
         ↓
┌──────────────────────────────────────┐
│ Service                              │
│ - Business Logic                     │
│ - แปลง DTO ↔ Entity                 │
└──────────────────────────────────────┘
         ↓
┌──────────────────────────────────────┐
│ Repository                           │
│ - CRUD operations                    │
│ - ใช้ Entity คุยกับ DB              │
└──────────────────────────────────────┘
         ↓
┌──────────────────────────────────────┐
│ Model/Entity (User, Order, Product)  │ ← โครงสร้างของ Table
│ = โครงสร้างของ Database Table       │
└──────────────────────────────────────┘
         ↓
┌──────────────────────────────────────┐
│ Database (MySQL/PostgreSQL)          │
│ - users table                        │
│ - orders table                       │
│ - products table                     │
└──────────────────────────────────────┘
